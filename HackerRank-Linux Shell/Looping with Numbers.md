# Looping with Numbers

## Task
Use a for loop to display the natural numbers from 1 to 50.  

## Procedure
1. Declare a for loop
for i in {start..end..increment}
do
  echo 
done
2. Make the for loop print out the numbers from the range 1 to 50: 
```
for i in {1..50..1}
do
  echo $i
done
```

## Solution
```
for i in {1..50..1}
do
  echo $i
done

```
